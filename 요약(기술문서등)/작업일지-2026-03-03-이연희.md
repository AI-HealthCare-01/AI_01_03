### 2.2 OCR API/서비스 구현
- OCR 설정 로딩 확장
  - `.env` + `../env` 동시 로딩
  - `OCR_SECRET_KEY`, `OCR_INVOKE_URL` 설정 필드 추가
  - 파일: `app/core/config.py`
- Clova OCR 호출 서비스 구현
  - 이미지 URL 기반 OCR API 호출
  - OCR 결과 텍스트 수집 및 처방 문장 파싱(정규식)
  - 파일: `app/services/ocr.py`
- OCR 요청 스키마 확장
  - `image_url`, `user_id`, `save_to_db` 필드 추가
  - 파일: `app/dtos/integration.py`
- OCR 라우터 고도화
  - `text` 또는 `image_url` 입력 처리
  - 파싱 실패 시 `PARSE_FAILED`
  - `save_to_db=true`일 때 DB 저장 시도
  - 저장 실패 시 `OCR_DB_SAVE_FAILED`
  - 파일: `app/apis/integration_routers.py`
